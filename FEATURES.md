# Features

## Resume Upload

Users can upload resumes in PDF format.

## Skill Extraction

The system identifies skills such as:

* Python
* Java
* SQL
* HTML
* CSS
* Machine Learning

## Resume Score

Generates a score out of 100 based on skills and keywords.

## Suggestions

Provides suggestions to improve resume quality.

## Future Scope

* ATS Compatibility Checker
* AI Career Recommendations
* Resume Summary Generator
